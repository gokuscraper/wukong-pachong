# wukong-pachong

悟空爬虫是一家专注于数据采集、爬虫定制与结构化交付的工作室。

悟空爬虫不是模板平台，也不是通用型SaaS产品，而是一个以“结果交付”为导向的数据服务工作室。 

我们关注的不只是“能不能采到数据”，更关注“交付的数据是否干净、规范、可直接使用”。
